# Selector Strategies

Write resilient, accessible selectors that survive UI changes and promote good testing habits.

## What You'll Learn

- The selector priority hierarchy (getByRole, getByLabel, getByTestId, etc.)
- Why accessibility-first selectors lead to more stable tests
- How to avoid brittle CSS and XPath selectors
- Strategies for handling dynamic content and complex DOM structures

See [SKILL.md](./SKILL.md) for the full guide. Follow the **Selector Priority Hierarchy** to choose the right selector for every situation.
