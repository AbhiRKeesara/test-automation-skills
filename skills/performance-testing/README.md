# Performance Testing

Measure and assert on application performance using Playwright for Web Vitals, network timing, and resource loading.

## What You'll Learn

- How to measure Core Web Vitals (LCP, FID, CLS) with Playwright
- Patterns for network performance and resource loading analysis
- Setting up performance budgets and automated assertions
- Using Playwright tracing and profiling for performance investigations

See [SKILL.md](./SKILL.md) for the full guide. Start with the **Web Vitals Measurement** section to capture your first performance metrics.
